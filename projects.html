CS50
CS50 is a free online course from Harvard University that teaches programming languages and web development. It is a course that teaches students to code by building projects such as in C, Python, SQL, HTML, CSS, and JavaScript.
This website, per example, was built as the last project of this course.

It is a 11 week course in which you are expected to work from 10-20 hours per week, which makes up to up to 220 hours in total, plus the final project.

The objective of beginning this course was to take some advance for some of my uni classes, but I ended up liking it so much that in the end I actually learned way more than what was needed.
